# MoodAura
#  MoodAura (still learning )

Hi!
This is my small project called **MoodAura**. I made this to try and help people feel a little better when they are sad or stressed.

I’m still learning web development, so this project is not perfect, but I tried my best to make something meaningful.

---

## What this project does

* Shows random quotes depending on mood

* Has different sections like:

  1.depressed
  2.tired
  3.happy
  4.demotivated
  5.bored

* Small interactive things like:

  * Background color changes
  * Hearts / animations 
  * Buttons like *“I’m fine”* and *“Still sad”*

---

##  What I tried here

In this project, I tried to create around 5 different mood sections with their own small interactions and feel.

Each section behaves a bit differently (quotes, colors, effects, etc.), but I know it’s not perfect yet.

I experimented with:

* Multiple mood-based sections
* Continuous/random quotes
* Simple animations like falling hearts
* Buttons that react differently

Some things worked, some didn’t  — but I learned a lot while building this.
